# Mini-WhatsApp
This is a basic web-based Mini WhatsApp Chat App that allows users to perform CRUD (Create, Read, Update, Delete) operations on chat messages. Data is stored in an array for simplicity.

Mini WhatsApp Clone 💬

This is a simple WhatsApp-style real-time chat application built using HTML, CSS, JavaScript, and Firebase Realtime Database.

🚀 Features

- Real-time chat
- Username login
- Chat bubble UI like WhatsApp
- Message timestamps
- Responsive design

🛠 Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Realtime Database

📌 How It Works

Users enter their username and start chatting. Messages are stored in Firebase and appear instantly for all users connected to the application.

🎯 Purpose

This project was created to practice JavaScript, Firebase integration, and real-time web application development.
